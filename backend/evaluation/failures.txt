Cand (75): Describe waitKey in cv2 in Python with proper code examples and outputs. 

waitKey in OpenCV is a function used to wait for a specified time for a keyboard event or until a key is pressed. It is used in conjunction with the displayImage function to display an image on the screen and wait for user input. 

The function takes one argument, which is the time in milliseconds to wait before returning. If the user presses a key during this time, the function will return the ASCII value of the key pressed. If no key is pressed, the function will return -1.

Here is an example of how to use waitKey in Python:

```python
import cv2

# Create a window
cv2.namedWindow('image')

# Display an image
img = cv2.imread('image.jpg')
cv2.imshow('image', img)

# Wait for a key press
key = cv2.waitKey(0)

# Check if the 'q' key was pressed
if key == ord('q'):
    print('You pressed the "q" key')
else:
    print('You pressed the "{}" key'.format(chr(key)))

# Close the window
cv2.destroyAllWindows()
```

In this example, the program will display an image on the screen and wait for the user to press a key. If the user presses the 'q' key, the program will print a message saying that the 'q' key was pressed. If the user presses any other key, the program will print the ASCII value of the key pressed.

Here is another example that uses waitKey to display a video and wait for the user to press a key:

```python
import cv2

# Create a window
cv2.namedWindow('video')

# Open a video file
cap = cv2.VideoCapture('video.mp4')

while True:
    # Read a frame from the video
    ret, frame = cap.read()

    # Display the frame
    cv2.imshow('video', frame)

    # Wait for a key press
    key = cv2.waitKey(1)

    # Check if the 'q' key was pressed
    if key == ord('q'):
        break

# Release the video capture object
cap.release()

# Close the window
cv2.destroyAllWindows()
```

In this example, the program will display a video on the screen and wait for the user to press a key. If the user presses the 'q' key, the program will break out of the loop and close the window.
